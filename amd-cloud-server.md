## Câu hỏi thường gặp

AMD Cloud Server khác gì VPS thường? __

Cloud Server chạy trên hạ tầng HA với hiệu năng ổn định, tự động failover khi sự cố. Hệ thống lưu trữ phân tán NVMe Enterprise với Replica 3 đảm bảo an toàn dữ liệu tối đa. VPS thường chạy trên single node với lưu trữ cục bộ, có rủi ro nếu không có cơ chế bảo vệ dữ liệu tốt.

Backup được lưu trữ bao lâu? __

Backup tự động hàng tuần và lưu trữ 2 bản gần nhất. Có thể restore bất kỳ bản backup nào trong 2 tuần gần nhất.

High Availability hoạt động như thế nào? __

Hệ thống Cloud Server được xây dựng trên cụm nhiều máy chủ vật lý. Khi một máy chủ vật lý gặp sự cố phần cứng, hệ thống sẽ tự động phát hiện và chuyển các máy chủ ảo sang máy chủ vật lý khác trong cụm. Quá trình này diễn ra tự động trong vài phút mà không cần sự can thiệp của kỹ thuật viên, giúp giảm thiểu thời gian gián đoạn dịch vụ.

Có hỗ trợ private network không? __

AZDIGI không hỗ trợ private network mặc định trên dịch vụ AMD Cloud Server, tuy nhiên bạn có thể liên hệ bộ phận kỹ thuật nếu có nhu cầu.

Windows Server có sẵn license không? __

Các template Windows của AZDIGI không hỗ trợ sẵn bản quyền mà được cài phiên bản dùng thử, người dùng cần tự trang bị giấy phép sử dụng.

AZDIGI có hỗ trợ chuyển dữ liệu VPS không? __

AZDIGI hỗ trợ chuyển dữ liệu miễn phí từ VPS nhà cung cấp khác về VPS của AZDIGI. Tuy nhiên, chỉ hỗ trợ chuyển các dữ liệu liên quan đến website và giới hạn 3 website/ngày. Vui lòng liên hệ bộ phận kỹ thuật để được hỗ trợ.
