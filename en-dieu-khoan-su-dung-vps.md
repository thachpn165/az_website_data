## VPS Service Description
AZDIGI VPS (Virtual Private Server) provides virtual servers with dedicated resources:

  * Dedicated CPU, RAM, SSD - not shared
  * Full root access / Administrator
  * Multiple OS support: CentOS, Ubuntu, Windows
  * Dedicated IPv4
  * 99.9% uptime

2

## Resources & Limits
Each VPS package has fixed resources:

  * **CPU:** 1 to 16 vCPU cores
  * **RAM:** 1GB to 64GB
  * **Storage:** NVMe SSD 20GB to 500GB
  * **Bandwidth:** Unlimited

3

## User Responsibility
With VPS, customers have full control and responsibility for:

  * Installing and configuring OS, software
  * Server security: firewall, SSH keys, updates
  * Backing up important data
  * Not using for illegal purposes
  * Not performing network attacks, DDoS, spam

AZDIGI only supports infrastructure layer, not application installation/configuration.

4

## Root Access Rights
VPS provides full root/administrator access:

  * Full rights to install any software
  * Customize kernel, network settings
  * Install control panels: cPanel, Plesk, DirectAdmin

**Note:** With root access, customers are fully responsible for VPS security and operation.

5

## VPS Technical Support Policy
AZDIGI VPS is an _unmanaged_ service, however we commit to technical support as follows:

  * Support installing control panels or auto webserver scripts for PHP and MySQL setups
  * No support for manual webserver installation or custom configurations
  * Support migrating up to **3 websites** PHP/MySQL (WordPress, Joomla, Drupal...) to AZDIGI VPS
  * No guarantee for smooth migration if issues arise from old provider or source code
  * No support for fixing website errors caused by source code
  * Customers are responsible for website security; AZDIGI does not support malware/virus cleanup or admin access recovery

## Security & Backup
### Security
  * Customers manage their own VPS security
  * Change default password immediately after receiving VPS
  * Install and configure firewall
  * Update OS regularly

### Backup
  * AZDIGI performs weekly backup, keeping 2 most recent copies
  * Customers are encouraged to backup externally
