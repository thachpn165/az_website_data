## Câu hỏi thường gặp

Website tạo bằng Cursor/Bolt/v0 có thể host ở đâu? __

Tùy framework: Astro/React/Next.js static có thể deploy trên Pro Hosting. WordPress cần PHP hosting. Node.js app cần VPS hoặc hosting có Node.js support.

Nên chọn Hosting hay VPS? __

Hosting phù hợp cho website mới bắt đầu với chi phí thấp. VPS phù hợp khi website có nhiều traffic hoặc cần cấu hình đặc biệt. AZDIGI hỗ trợ nâng cấp dễ dàng.

AZDIGI có hỗ trợ deploy không? __

Có, đội ngũ kỹ thuật hỗ trợ cài đặt và cấu hình miễn phí. Bạn chỉ cần cung cấp source code, chúng tôi sẽ giúp deploy lên hosting/VPS.

Website có traffic cao thì làm sao? __

AZDIGI luôn sẵn sàng giải pháp mở rộng. Bạn có thể nâng cấp lên VPS với tài nguyên riêng, hoặc Cloud Server cho traffic rất cao. Đội ngũ tư vấn sẽ giúp bạn chọn gói phù hợp.
