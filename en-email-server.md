## Frequently Asked Questions
How is Email Server different from Email Hosting? __

Email Server uses a dedicated server with larger resources, unlimited email addresses and high-speed processing for large volumes. Suitable for businesses needing 100+ email addresses.

What is Backup Relay? __

Backup Relay is an independent external backup mail server that optimizes outgoing mail delivery, reduces spam folder placement and protects IP reputation. Available from Email Server 2 plan.

Is Email Server backed up? __

Yes. Email data is automatically backed up weekly and stored in 2 different versions. You can restore or download backups anytime.

Can I use multiple domains? __

Yes. Email Server has no domain limit. You can use any number of domains on same email server.

Which apps are compatible? __

Email Server is compatible with all devices and email apps like Outlook, Airmail, Mail iOS, and other mobile email applications.
