## Frequently Asked Questions
How is X-Platinum VPS different from regular VPS? __

X-Platinum VPS uses new generation Intel Xeon Platinum CPUs (Gen 1 & Gen 2) instead of older CPU models. Enterprise NVMe storage with significantly faster read/write speeds compared to standard SSDs and HDDs. CPU/RAM resources are allocated separately, never oversold, ensuring stable performance even when other VPS on the same node are under high load.

Do I have full administrative access to the VPS? __

Yes. You get full root access (Linux) or Administrator (Windows), with complete authority to install, configure, and manage your VPS like a physical server. You can install any software, control panel, or configure firewall according to your specific needs.

How long does VPS deployment take after payment? __

VPS is automatically activated immediately after successful payment, usually within 1-5 minutes. You will receive an email containing VPS login information (IP, username, password) and can start using it immediately via SSH or Remote Desktop.

What operating systems can I install on X-Platinum VPS? __

X-Platinum VPS supports multiple operating systems: Ubuntu (20.04, 22.04, 24.04), AlmaLinux 8/9, Debian 11/12, CentOS 7, Windows Server 2019/2022. You can reinstall OS anytime via Virtualizor panel without technical support.

How does backup work and can I restore? __

AZDIGI automatically backs up your VPS and stores 2 weekly backups. You can restore anytime via the Virtualizor control panel or contact support for assistance. However, we recommend proactively configuring backup within your VPS to store data externally (cloud storage, FTP, etc.) for maximum safety. If you need consultation on suitable backup solutions, contact support - completely free of charge.

Can I upgrade or downgrade the VPS plan? __

Yes. You can upgrade to a higher plan anytime, the system will calculate the price difference based on remaining time. Downgrading requires contacting support for appropriate advice. If your needs exceed X-Platinum VPS, you can upgrade to Cloud Server or Dedicated Server.

What technical support does AZDIGI provide for X-Platinum VPS? __

AZDIGI provides 24/7 technical support via ticket, live chat, and hotline. Support scope includes: infrastructure troubleshooting, basic configuration guidance, free migration support from other providers, performance optimization consulting. Average response time is under 15 minutes for urgent requests.

Does AZDIGI provide VPS monitoring support? __

AZDIGI supports technical issues related to infrastructure and basic usage guidance. However, monitoring your VPS and software/applications running inside the VPS is the user's responsibility. You should set up a monitoring system (Uptime Kuma, Zabbix, etc.) to track your VPS status. If you need consultation on suitable monitoring solutions, contact our technical team for assistance.

What is the refund and cancellation policy? __

AZDIGI guarantees 100% refund within the first 30 days if you are not satisfied with the service, no reason required and no fees retained. After 30 days, you can cancel anytime, remaining fees will be refunded proportionally based on unused time (excluding promotional discounts).

Does AZDIGI support VPS data migration? __

AZDIGI provides free data migration from other VPS providers to AZDIGI VPS. However, only website-related data is supported with a limit of 3 websites per day. Please contact technical support for assistance.
