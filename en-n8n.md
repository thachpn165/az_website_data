## Frequently asked questions about n8n
How much RAM do I need to run n8n? __

Minimum 1GB RAM for basic n8n. Recommended 2-4GB RAM if running many workflows or processing large data. For AI/ML workflows, choose 8GB RAM or more.

Is n8n free? __

n8n is open-source and completely free. You only pay for VPS hosting. No workflow limits, no execution limits like n8n cloud plans.

What services can I connect with n8n? __

n8n supports 400+ integrations including: Google (Gmail, Sheets, Drive), Slack, Telegram, Discord, Notion, Airtable, MySQL, PostgreSQL, any HTTP API, and more.

Is SSL auto-installed? __

Yes, Let's Encrypt SSL is auto-configured after n8n installation. Just ensure hostname is a valid domain/subdomain pointed to VPS IP.

Where is n8n data stored? __

n8n data (workflows, credentials, execution history) is stored in SQLite or PostgreSQL database on your VPS. You have full control over your data.

Can I upgrade VPS later? __

Yes, you can upgrade VPS anytime when you need more resources. Upgrade process is quick and doesn't affect n8n data.
