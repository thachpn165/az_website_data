## Frequently Asked Questions
How is AMD Cloud Server different from regular VPS? __

Cloud Server runs on HA infrastructure with stable performance and automatic failover. Distributed NVMe Enterprise storage with Replica 3 ensures maximum data safety. Regular VPS runs on single node with local storage, which has risks without proper data protection.

How long are backups retained? __

Weekly automatic backups stored for 2 most recent copies. You can restore any backup within the last 2 weeks.

How does High Availability work? __

Cloud Server system is built on a cluster of multiple physical servers. When a physical server encounters hardware failure, the system automatically detects and migrates virtual servers to another physical server in the cluster. This process happens automatically within minutes without technician intervention, minimizing service downtime.

Is private network supported? __

AZDIGI does not support private network by default on AMD Cloud Server, however you can contact technical support if needed.

Is Windows Server license included? __

AZDIGI's Windows templates do not include license and are installed as trial versions. Users need to provide their own license.

Does AZDIGI support VPS data migration? __

AZDIGI provides free data migration from other VPS providers to AZDIGI VPS. However, only website-related data is supported with a limit of 3 websites per day. Please contact technical support for assistance.
