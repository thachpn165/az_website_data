## Câu hỏi thường gặp

VPS nào phù hợp để chạy phần mềm kế toán MISA? __

MISA cần tối thiểu 4GB RAM và Windows Server. Khuyên dùng VPS từ 8GB RAM cho team 5-10 kế toán viên, hoặc 16GB+ cho team lớn hơn.

Phần mềm kế toán trên VPS có an toàn không? __

Rất an toàn. Dữ liệu lưu trên VPS riêng của bạn, mã hóa SSL, firewall bảo vệ. An toàn hơn so với lưu trên máy tính cá nhân.

Có thể truy cập phần mềm kế toán từ điện thoại không? __

Có, với các phần mềm web-based như Akaunting. Với MISA hoặc Fast, có thể dùng Remote Desktop từ điện thoại.

AZDIGI có hỗ trợ cài đặt phần mềm kế toán không? __

Có, chúng tôi hỗ trợ cài đặt Windows Server, SQL Server và các phần mềm mã nguồn mở. Với MISA/Fast cần license từ nhà cung cấp.
