## Phạm Vi Hỗ Trợ

### Hỗ trợ miễn phí:

  * Vấn đề về hạ tầng, network, server
  * Cài đặt lại hệ điều hành VPS
  * Khôi phục backup (khi có sẵn)
  * Hướng dẫn sử dụng dịch vụ
  * Cài đặt SSL lần đầu



### Không hỗ trợ:

  * Lập trình, sửa code website
  * Cài đặt/cấu hình ứng dụng trên VPS
  * Tối ưu website, database
  * Vấn đề bảo mật do code khách hàng

## Quy Trình Leo Thang (Escalation)

AZDIGI áp dụng quy trình hỗ trợ 4 tầng để đảm bảo mọi vấn đề luôn được xử lý triệt để:

  1. **Level 1:** Kỹ thuật viên hỗ trợ
  2. **Level 2:** Supervisor/Team Lead (yêu cầu qua ticket)
  3. **Level 3:** Manager (email: manager@azdigi.com)
  4. **Level 4:** Ban Quản Trị công ty
