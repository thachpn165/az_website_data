## Frequently Asked Questions

How is affordable Pro VPS different from X-Platinum VPS? __

Pro VPS is an affordable VPS balanced between performance and cost, using Enterprise SSD with 99.9% uptime, suitable for small websites, dev/test, learning. X-Platinum VPS has new generation Platinum CPU, premium NVMe, priority 24/7 support and snapshot, suitable for businesses.

Do I have full administrative access to affordable VPS? __

Yes. You get full root access (Linux) or Administrator (Windows), with complete authority to install, configure, and manage your VPS like a physical server. You can install any software, control panel, or configure firewall as needed.

How long does affordable VPS deployment take? __

VPS is automatically activated immediately after successful payment, usually within 5 minutes. You will receive an email containing VPS login information (IP, username, password) and can start using it immediately.

How does affordable VPS backup work? __

All Pro VPS plans have automatic weekly backup. You can restore via Virtualizor panel or contact support. We recommend configuring external backup for maximum safety.

Can I upgrade affordable VPS to a higher plan? __

Yes. You can upgrade within Pro VPS line or upgrade to X-Platinum VPS anytime. AZDIGI provides free migration and configuration.

Does AZDIGI provide technical support for affordable VPS? __

Yes. AZDIGI provides technical support via ticket and live chat. Support scope includes: infrastructure troubleshooting, basic configuration guidance, migration support from other providers.

Does AZDIGI provide monitoring support for affordable VPS? __

AZDIGI supports technical issues related to infrastructure. Monitoring VPS and software inside is user's responsibility. You can contact support for consultation on suitable monitoring solutions.

What is the refund policy for affordable VPS? __

AZDIGI guarantees refund within the first 7 days if you are not satisfied with the affordable VPS service, no reason required. After 7 days, remaining fees are refunded proportionally based on unused time.

Is affordable VPS suitable for production? __

Pro VPS is suitable for small websites, blogs, landing pages with moderate traffic. If you need production for large websites, e-commerce, or critical applications, choose X-Platinum VPS for higher performance and uptime.

Does AZDIGI support VPS data migration? __

AZDIGI provides free data migration from other VPS providers to AZDIGI VPS. However, only website-related data is supported with a limit of 3 websites per day. Please contact technical support for assistance.
